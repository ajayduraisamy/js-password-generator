# JavaScript Password Generator

A responsive password generator built using **HTML, CSS, Bootstrap, and Vanilla JavaScript**.

The application generates secure passwords based on user preferences and provides real-time strength evaluation.

---

## Features

- Generate random secure passwords
- Length customization (6â€“30 characters)
- Uppercase, number, and symbol options
- Copy-to-clipboard button
- Password strength indicator (Weak / Medium / Strong)
- Clean Bootstrap-based UI

---

## Usage

1. Clone the repository
2. Open `index.html` in any browser
3. Set your options and click **Generate Password**
4. Use **Copy to Clipboard** to copy the password
